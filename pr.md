##  Description
This comprehensive HR Onboarding Automation Dashboard built entirely using Airtable's native capabilities. The solution transforms raw onboarding data into a structured workflow for HR operations.

###  Task 1: Data Cleaning and Categorization
- Formatted and concatenated employee names for reliable identification.
- Cleaned and standardized email inputs natively using `LOWER()` and `TRIM()`.
- Added status categorization to automatically classify hires into "High Value", "Medium Value", and "Low Value" priorities based on equipment spend.
- Implemented formula-based tracking for days spent in the onboarding pipeline.

###  Task 2: Automation and HR Dashboard
- **Automation Logic**: Created workflows to detect "High Value" hires. Includes conditional branching to notify HR and safely log records into an audit tracking table, alongside alerting logic for missing emails.
- **HR Dashboard**: Designed an Airtable Interface with summary KPI tiles, filtered grid views, and detailed records mapped to specific candidates for immediate HR review.
